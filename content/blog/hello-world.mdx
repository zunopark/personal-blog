안녕하세요! Next.js 기반의 개인 블로그에 오신 것을 환영합니다. 🎉

이것은 첫 번째 테스트 게시물입니다. 마크다운으로 작성되었으며, **MDX**를 사용하여 렌더링됩니다.

<Callout type="info">
이 블로그는 Next.js 16, React 19, TypeScript로 구축되었습니다. MDX를 통해 커스텀 컴포넌트를 사용할 수 있어요!
</Callout>

## 주요 기능

이 블로그는 다음과 같은 특징을 가지고 있습니다:

- **Markdown/MDX 기반** - 글 작성이 간편합니다
- **SEO 최적화** - 검색 엔진 노출에 최적화되어 있습니다
- **Apple 스타일 디자인** - 깔끔하고 모던한 UI
- **커스텀 컴포넌트** - Callout, AdSlot 등 다양한 컴포넌트 지원

### 코드 하이라이팅

JavaScript 코드를 아름답게 표시할 수 있습니다:

```javascript
// Next.js 블로그 예제
console.log("Hello from Next.js blog!");

const greeting = (name) => {
  return `안녕하세요, ${name}님!`;
};

greeting("개발자");
```

<Callout type="warning">
이 프로젝트는 아직 개발 중입니다. 일부 기능은 추후 추가될 예정입니다.
</Callout>
